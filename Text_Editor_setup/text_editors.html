Sublime 2

Navigate to the following link:

http://www.sublimetext.com/2

You'll notice several options under the Download Header Download the file associated with your operating system Follow the download instructions and use all defaults. Make note of the file location

You'll also notice a sign saying that you can download and "evaluate it for free",

however a license must be purchased for continued use. You can extend your trial period

indefinitely, so don't worry about it. I think I have renewed it 2-3 dozen times.

Atom

Navigate to the following link

https://atom.io/

At the top of the page you'll notice it has a big red button that says "Download Windows Installer". If you have a windows machine, click it and follow the default download steps.

If you don't have a windows machine, look right below the red button for the phrase "Other platforms" You will then find a page with the header "downloads". Download the appropriate file for your system Follow the default download steps. Make note of the file location where Atom is downloaded to.
